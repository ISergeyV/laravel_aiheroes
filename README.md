# Mr. EuroFix | Handyman CRM
 
 Система управления взаимоотношениями с клиентами (CRM) и лидами для сервиса "Mr. EuroFix".
 
 ---
 
 ## 1. Технологический стек
 
 Этот проект построен на современном и производительном наборе технологий, выбранных для обеспечения надежности, масштабируемости и удобства разработки.
 
 ### 1.1. Серверная часть (Backend)
 *   **Язык программирования**: `PHP 8.3+`
 *   **Фреймворк**: `Laravel 12+`
 *   **Архитектурные компоненты**:
     *   **Eloquent ORM**: Для объектно-реляционного отображения и взаимодействия с базой данных.
     *   **Laravel Queues**: Для асинхронного выполнения фоновых задач (например, обработка AI-запросов, отправка уведомлений).
 *   **Интеграции**:
     *   **AI Services**: Интеграция с внешними сервисами искусственного интеллекта через API для обработки и анализа данных.
 
 ### 1.2. Панель администрирования (UI/UX)
 *   **Основной инструмент**: `Filament v3`
 *   **TALL Stack**:
     *   `Tailwind CSS`: Utility-first CSS-фреймворк для создания пользовательского интерфейса.
     *   `Alpine.js`: Минималистичный JavaScript-фреймворк для интерактивности.
     *   `Livewire`: Full-stack фреймворк для создания динамических интерфейсов на PHP.
 
 ### 1.3. Ключевые библиотеки
 *   `spatie/laravel-settings`: Управление настройками приложения с сохранением в БД.
 *   `filament/spatie-laravel-settings-plugin`: UI-интеграция для управления настройками в панели Filament.
 *   `doctrine/dbal`: Библиотека для расширенных манипуляций со схемой базы данных в миграциях.
 
 ### 1.4. Среда разработки и инструментарий
 *   **IDE**: `PHPStorm`
 *   **Локальное окружение**: `DDEV` (на базе `Docker`)
 *   **Операционная система**: `WSL2` (Ubuntu 24.04)
 *   **Управление зависимостями**: `Composer`
 
 ### 1.5. Инфраструктура (Production/Staging)
 *   **Веб-сервер**: `Nginx` / `Apache`
 *   **База данных**: Реляционная СУБД (`MySQL`/`MariaDB`/`PostgreSQL`)
