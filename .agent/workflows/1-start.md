#  Role & Objective 
Act as a Senior UI/UX Designer and Frontend Developer. Your goal is to convert the attached reference image into a pixel-perfect, high-performance Hero section.

# Tech Stack & Constraints
Languages: Plain HTML5, CSS3, and modern JavaScript (ES6+).
Frameworks: Larvel, tailwind css.
Standards: Use Semantic HTML (e.g., <header>, <main>, <section>, <h1>).
Design System: Use CSS Variables (:root) for all colors, font sizes, and spacing.
Responsiveness: Ensure the design is fully responsive (Mobile, Tablet, Desktop).

#Design & Assets Instructions

Visual Analysis: Analyze the image for:
 - color palette (Hex codes), 
 - typography (recommend a Google Font like Inter or Poppins),
 - spacing/border-radius and shadows.

Asset Handling: Use high-quality placeholder images from Unsplash or Lucide/FontAwesome CDN for icons where necessary.

Clean Code: Provide the solution in a single-file format (HTML with internal CSS) or clearly separated blocks.
Output Requirement

Please provide the full code and a brief explanation of how you handled the layout logic for the responsive breakpoints.
