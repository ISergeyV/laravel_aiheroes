# Скрипт-помощник: Авто-конвертация для Laravel

Этот воркфлоу оптимизирует рутинный процесс переноса готовой HTML-верстки в структуру Laravel, заменяя кастомный CSS на утилитарные классы Tailwind CSS.

## Инструкция для ИИ (Агента)

При вызове этого сценария выполни следующие шаги для указанного пользователем HTML-кода (или файла):

1. **Анализ зависимостей CSS:** 
   Сначала найди все классы в предоставленном HTML-блоке и просмотри соответствующие стили в файле `/style.css`. Изучи переменные (colors, fonts, spacing).
   
2. **Конвертация в Tailwind:**
   Переведи найденные CSS-правила в эквиваленты Tailwind CSS (версии 3.x+). 
   *Пример:* `display: flex; justify-content: center;` -> `flex justify-center`.
   
3. **Адаптация под Blade:**
   - Помести результат в `resources/views/components/` (или предложи пользователю создать этот файл, если папки еще нет).
   - Если в блоке есть повторяющийся контент (список карточек услуг, кейсов, иконок), автоматически преврати его в структуру с `@foreach ($items as $item)` и извлеки данные в массив.
   - Замени статические пути к картинкам на `{{ asset('...') }}`.

4. **Очистка (Clean-up):**
   Предложи удалить исходные CSS-правила из `style.css`, так как они больше не нужны (они заменены классами Tailwind прямо в HTML).

5. **Отчет (Фидбек):**
   Кратко покажи пользователю:
   - Новый код Blade компонента.
   - Стили, которые были удалены.

> **Как запустить:** Напиши ИИ в чат `/GEMINI [Название блока/ID]`, например: `/GEMINI блок Feature Grid`.
